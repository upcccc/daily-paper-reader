## 摘要
大语言模型在推荐系统中取得了巨大成功。然而，用户数据的局限性和稀疏性往往限制了大语言模型有效建模行为模式的能力。为解决这一问题，现有研究通过执行跨域推荐（CDR）任务探索了跨域解决方案。但以往的方法通常假设领域之间存在重叠且可以被轻易访问。目前尚无大语言模型方法解决跨域推荐场景下的隐私保护问题，即隐私保护跨域推荐（PPCDR）。利用大语言模型进行非重叠隐私保护跨域推荐具有挑战性，原因在于：1）无法跨域共享用户身份或行为数据，阻碍了有效的跨域对齐；2）跨域数据模态的异构性增加了知识集成的复杂性；3）将传统推荐模型的协同过滤信号与大语言模型融合较为困难，因为它们运行在不同的特征空间中。为解决上述问题，我们提出了 SF-UBM，一种语义增强的联邦用户行为建模方法。具体而言，针对挑战 1，我们利用自然语言作为通用桥梁，通过语义增强的联邦架构连接不相交的领域。在该架构中，基于文本的物品表示被加密并共享，而用户特定数据则保留在本地。针对挑战 2，我们设计了一个事实计数器知识蒸馏模块，以跨不同数据模态集成领域无关知识与领域特定知识。针对挑战 3，我们将预先学习的用户偏好和跨域物品表示投影到软提示空间中，对齐行为空间与语义空间，以实现有效的大语言模型学习。我们在三对真实世界领域上进行了广泛的实验，实验结果证明了 SF-UBM 与近期最先进方法相比的有效性。

## Abstract
Large Language Models have shown great success in recommender systems. However, the limited and sparse nature of user data often restricts the LLM's ability to effectively model behavior patterns. To address this, existing studies have explored cross-domain solutions by conducting Cross-Domain Recommendation tasks. But previous methods typically assume domains are overlapped and can be accessed readily. None of the LLM methods address the privacy-preserving issues in the CDR settings, that is, Privacy-Preserving Cross-Domain Recommendation. Conducting non-overlapping PPCDR with LLM is challenging since: 1)The inability to share user identity or behavioral data across domains impedes effective cross-domain alignment. 2)The heterogeneity of data modalities across domains complicates knowledge integration. 3)Fusing collaborative filtering signals from traditional recommendation models with LLMs is difficult, as they operate within distinct feature spaces. To address the above issues, we propose SF-UBM, a Semantic-enhanced Federated User Behavior Modeling method. Specifically, to deal with Challenge 1, we leverage natural language as a universal bridge to connect disjoint domains via a semantic-enhanced federated architecture. Here, text-based item representations are encrypted and shared, while user-specific data remains local. To handle Challenge 2, we design a Fact-counter Knowledge Distillation module to integrate domain-agnostic knowledge with domain-specific knowledge, across different data modalities. To tackle Challenge 3, we project pre-learned user preferences and cross-domain item representations into the soft prompt space, aligning behavioral and semantic spaces for effective LLM learning. We conduct extensive experiments on three pairs of real-world domains, and the experimental results demonstrate the effectiveness of SF-UBM compared to the recent SOTA methods.

以下是对该论文的深度结构化总结：

### 1. 核心问题与研究动机
*   **核心问题**：如何在保护用户隐私的前提下，利用 LLM 强大的语义理解能力进行**非重叠跨域推荐（Non-overlapping PPCDR）**。
*   **研究背景**：
    *   LLM 在推荐系统中表现优异，但受限于单域数据的稀疏性。
    *   现有的跨域推荐（CDR）方法通常假设领域间有重叠用户/物品，且忽略了数据隐私。
    *   **三大挑战**：
        1.  **隐私壁垒**：无法直接共享用户身份或行为数据，导致跨域对齐困难。
        2.  **模态异构**：文本语义与用户 ID 模态难以有效集成。
        3.  **空间鸿沟**：传统协同过滤（CF）信号与 LLM 的语义特征空间不一致。

### 2. 方法论：SF-UBM 框架
SF-UBM 分为两个主要阶段：

#### 第一阶段：联邦跨域语义学习（Federated Cross-domain Semantic Learning）
*   **语义提取与加密**：利用 SBERT 提取物品文本嵌入。为保护隐私，提出一种**替换扰动机制**：先加入高斯噪声，再通过“相似性替换”策略（用最相似的物品向量替换原向量）来模糊原始特征，随后上传至服务器。
*   **领域无关语义挖掘**：服务器对各域上传的加密向量进行 **K-means++ 聚类**，提取跨域共享的语义质心，并下发给各客户端作为全局知识。
*   **事实计数器知识蒸馏（FKD）**：在本地客户端，通过知识蒸馏将全局语义知识注入 ID 编码层。利用“事实”用户偏好（真实序列）与“反事实”偏好（随机序列）的一致性约束，对齐 ID 模态与文本模态。

#### 第二阶段：基于 LLM 的行为预测
*   **令牌空间知识注入**：设计两个投影层（Projectors），将预训练的**用户表示**（来自 SASRec）和**增强后的物品嵌入**投影到 LLM 的软提示（Soft Prompt）空间。
*   **混合提示微调**：将投影后的连续向量与静态文本指令结合，输入到 **OPT-6.7B** 模型中，通过预测下一个物品的 Token 进行微调。

### 3. 实验设计
*   **数据集**：使用了 Amazon（Health-Beauty, Food-Kitchen）和 Amazon-Movielens（Books-Movielens）三对真实世界领域。所有领域均处理为用户和物品完全不重叠。
*   **评估指标**：Hit Ratio @1（命中率）和 Valid Ratio（输出格式有效率）。
*   **对比方法（Baselines）**：
    *   **纯 ID 类**：SASRec, GRU4Rec。
    *   **跨域类**：PTUPCDR, RecGURU。
    *   **ID+文本类**：UniSRec, PFCR, FFMSR（后两者为非重叠 PPCDR 的 SOTA）。
    *   **LLM 类**：TALLRec, A-LLMRec, MLP-LLM。

### 4. 资源与算力
*   **硬件设备**：主要实验在 **NVIDIA RTX 3090 GPU** 上完成；其中 TALLRec 基准测试在 **NVIDIA RTX 5090 GPU** 上运行。
*   **模型规模**：LLM 主干网络采用 **OPT-6.7B**。
*   **训练细节**：Stage 1 训练 10 个 epoch，Stage 2 映射训练 5 个 epoch，学习率设为 0.0001。

### 5. 实验数量与充分性
*   **实验规模**：在 3 个跨域场景下进行了全量对比实验。
*   **消融实验**：设计了 4 种变体（纯文本、纯 ID、无联邦学习、无 FKD 映射），验证了各模块的必要性。
*   **参数敏感性分析**：对聚类数量 $k$、损失权重 $\alpha, \beta$ 进行了详细调优。
*   **隐私分析**：专门测试了不同噪声水平 $\sigma$ 对推荐性能的影响，量化了隐私与效用之间的权衡。
*   **评价**：实验设计全面，涵盖了从性能对比到隐私安全性验证的各个维度，结果具有较高的说服力。

### 6. 主要结论
*   **性能领先**：SF-UBM 在所有数据集上的 Hit@1 均显著优于现有 SOTA 方法（如在 Books-Movielens 上比 A-LLMRec 提升了约 65%）。
*   **语义桥梁作用**：自然语言作为“通用语言”，在非重叠域之间起到了关键的对齐作用。
*   **隐私有效性**：提出的扰动机制在损失较小性能的前提下，能有效防止服务器推断原始用户特征。

### 7. 优点与亮点
*   **场景创新**：针对“非重叠”且“隐私保护”的 LLM 推荐这一极具挑战性的现实场景提出了系统性方案。
*   **多模态融合**：通过 FKD 巧妙地解决了 ID 信号与文本语义的对齐问题，避免了简单的拼接。
*   **软提示对齐**：利用投影层将传统推荐模型的协同过滤能力引入 LLM，实现了“行为空间”到“语义空间”的平滑过渡。

### 8. 不足与局限
*   **模态单一**：目前主要侧重于文本语义，尚未涉及图像、视频等多模态信息的跨域对齐。
*   **计算开销**：虽然使用了软提示投影，但微调 6.7B 规模的 LLM 对本地客户端（如果是真实联邦场景）仍有较高的算力要求。
*   **冷启动限制**：虽然缓解了数据稀疏，但对于完全没有任何历史行为的新用户，其依赖预训练 ID 模型的效果可能会受限。
